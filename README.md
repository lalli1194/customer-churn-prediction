
# Customer Churn Prediction

This project predicts whether a telecom customer will churn or not using Machine Learning and Flask.

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML/CSS

## Features
- Customer churn prediction
- Machine learning model deployment
- Flask web application
- User-friendly interface

## Project Structure

customer-churn-prediction/
│
├── app.py
├── model.pkl
├── requirements.txt
├── README.md
├── templates/
├── static/

## How to Run

1. Install requirements

pip install -r requirements.txt

2. Run Flask app

python app.py

3. Open browser

http://127.0.0.1:5000

## Machine Learning Algorithms Used
- Logistic Regression
- Random Forest
- Decision Tree
- XGBoost
